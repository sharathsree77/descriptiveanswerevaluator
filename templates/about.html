Features

Tutor Module 1: Authentication System

Authenticates the teachers. Allows only those with vaid login ID and password

Tutor Module 2: Create new tests

The teachers can create new exams and give the name of the exam

Student Module 1: Authentication System

Authenticates the students. Allows only those with valid USN and password

Student Module 2: Taking up of tests

Students select the test they wish to take up

Tutor Module 3: Question-Reference answer key entry

Teachers enter the number of questions. Also the question and the prescribed solutions for the same are entered

Tutor Module 4: Results

Teachers can check the results of every test. The marks of the students along with their USN, question, student's answers, marks awarded using various text-similarity algorithms and their total percentage score are displayed

Student Module 3: Answering to the questions

Students answer to the questions given by the tutor in the provided space

Student Module 4: Results

Once all the answers are submitted, students can immediately see their total score, individual scores awarded for each question and the reference answers along with their answers
